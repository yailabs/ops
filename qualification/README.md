# Qualification

`qualification/` contains executable qualification programs, waves, runs, and reports.

## Role

This layer proves runtime behavior against canonical law.
It does not define canonical normative policy.

## Contains

- qualification packs and templates
- qualification waves and run sets
- validation outputs and reports
- proof-oriented execution records

## Boundary

- Normative source: `law`
- Runtime implementation: `yai`
- Qualification/evidence publication: `ops`
