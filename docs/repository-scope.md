# ops repository scope

## Role

`ops` is the official bureau for qualification, evidence curation, publication, and collateral.

## In scope

- qualification runs and reports
- curated evidence and release bundles
- official publication package
- catalog/rollout/coverage visibility

## Out of scope

- normative law authorship
- runtime policy execution ownership
- runtime architecture authority
- dataplane/db/query implementation

## Boundary rule

`ops` material can describe and evidence behavior.
It does not define canonical runtime law.
