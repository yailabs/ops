# YAI Pilot Offer v1.0 (14 Days)

## Objective

Validate that one production-relevant AI workflow can be governed with deterministic containment, auditable evidence, and improved operational speed.

## Scope

- Scenario: SC102 governance grammar
- Baseline package: D1 + D8 Wave 1
- One customer workflow mapped to star case control objectives

## Deliverables

1. Configured runtime governance path
2. Executed qualification wave with evidence bundle
3. Verification result and report summary
4. Operator demo walkthrough
5. Expansion backlog proposal

## Customer prerequisites

- Technical owner and security owner assigned
- Target workflow and environment defined
- Access to required execution environment

## Success criteria

- Wave execution pass (`run-wave.sh` exit 0)
- Bundle verification pass (`verify.sh` exit 0)
- Prohibited effects blocked per baseline contract
- Evidence artifacts complete and reviewable

## Timeline

- Days 1-2: discovery + mapping
- Days 3-6: setup + baseline execution
- Days 7-10: workflow adaptation and reruns
- Days 11-13: reporting and stakeholder review
- Day 14: decision meeting (expand / defer)

## Commercial decision output

- Go-forward annual scope and commercial proposal
- phased rollout plan across additional domains/workflows
